# [hmr-cli](https://github.com/staven630/hmr-cli.git)
&emsp;&emsp;提供angular6以上HMR(热更新)功能

## 步骤
```
npm i -g hmr-cli
```
&emsp;&emsp;初始化angular项目,命令行进入该目录
```
hmr init
```
&emsp;&emsp;再次运行项目可以使用npm run hmr替代npm run serve
